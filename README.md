## 👨‍💻 Sobre mí

Soy un desarrollador enfocado en crear soluciones modernas, limpias y funcionales. Me especializo en el ecosistema de **Next.js** y **React**, diseñando interfaces que no solo se ven bien, sino que están optimizadas para el rendimiento y la escalabilidad.

Actualmente, estoy desarrollando sistemas de gestión administrativa (SIGACE) y herramientas financieras (RBO Finance), aplicando arquitecturas de bases de datos robustas y experiencias de usuario intuitivas.

---

## 🛠️ Stack Tecnológico

### Frontend

- **Frameworks:** React, Next.js (App Router)
- **Estilos:** Tailwind CSS (Modern Dark Mode, Glassmorphism)
- **Estado/Context:** React Context API, Custom Hooks

### Backend & Databases

- **Runtime:** Node.js
- **ORM:** Prisma
- **Bases de Datos:** MySQL (Relaciones complejas y optimización de queries), PostgresSQL
- **Autenticación:** NextAuth.js
